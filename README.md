# 工具集合项目

## 启动项目

### 1. 安装依赖
```bash
npm install
```

### 2. 启动项目

#### 同时启动前后端（推荐）
```bash
npm run dev:full
```

#### 分别启动前后端
```bash
# 启动后端服务器（端口3000）
npm run server

# 启动前端开发服务器（端口5173）
npm run dev
```

#### 仅启动后端
```bash
npm start
```

### 3. 访问地址
- 前端页面: http://localhost:8999
- 后端API: http://localhost:3000
